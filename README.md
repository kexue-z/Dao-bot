#

想要尝试部署吗？

那就先装好 docker 然后

下载 gocqhttp 到这个文件夹里 例如

`/dao-bot/go-cqhttp/go-cqhttp`

然后自己运行配置好 go-cqhttp 后保证能够登录 关闭 gocq

最后按照下面的运行一下

```
git clone https://github.com/kexue-z/Dao-bot.git
cd Dao-bot
chmod 755 run
./run
```

就好了
